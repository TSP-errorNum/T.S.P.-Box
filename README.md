# T.S.P.-Box
一个由被收费/登陆网站折磨疯了的普通大学生创建的立志解决数个如同BPM检测等问题的项目

**当前所有功能：**

>BPM检测
  
>视频提取音频
  
>(其他的还在来的路上)

------

**使用方式：**

>python BPMChak.py ./xxx.mp3
  
>python VideoToAudio.py xxx.mp4 yyy.mp3
